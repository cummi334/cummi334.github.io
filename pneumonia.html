Pneumonia Detection — Traditional ML Model Comparison

Comparative analysis of baseline machine learning classifiers for pneumonia detection, with emphasis on interpretability, training behavior, and practical tradeoffs between models rather than raw accuracy alone.

Project context

This project was completed as part of Computational Modeling and Data Analysis II. The goal was to build, compare, and critically evaluate multiple traditional machine learning models on a medical classification task, with attention to strengths, weaknesses, and limitations of each approach.

What I did

• Implemented and trained K-Nearest Neighbors, Logistic Regression, Perceptron, and Support Vector Machine classifiers using scikit-learn.
• Standardized features and carefully managed preprocessing to ensure fair comparisons across models.
• Evaluated models using accuracy and training time, highlighting computational and practical tradeoffs.
• Visualized results with matplotlib to communicate performance differences and model behavior.
• Identified limitations such as sensitivity to hyperparameters, interpretability challenges, and dataset constraints.

Key takeaways

This project reinforced the importance of baseline models in applied machine learning. While more complex models can offer performance gains, simpler classifiers often provide strong interpretability, faster training, and clearer failure modes. The comparison emphasized that model selection depends on context, data quality, and downstream constraints — not just headline accuracy.
