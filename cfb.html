Course Project

College Football Win% Prediction — Feature Engineering + Modeling

Built predictive models for team win percentage using cleaned college football data, engineered features, and improved model stability and interpretability by addressing multicollinearity.

R Data Cleaning Feature Engineering Regression Multicollinearity Visualization Model Evaluation
← Back to Projects Open PDF in new tab

Project context

Completed for Introduction to Data Science. The project focused on building an end-to-end modeling workflow in R: cleaning a real dataset, constructing useful features, fitting predictive models, and communicating findings with clear plots and responsible interpretation.

What I did

• Cleaned and enhanced team-level football data (handling missing values, standardizing variables, and validating joins/aggregations).
• Engineered features to capture team performance signals and improve predictive power.
• Built predictive models for win percentage and evaluated model fit and generalization behavior.
• Diagnosed multicollinearity and adjusted feature sets / model structure to improve stability and interpretability.
• Created visualizations in R to explain relationships between features and win percentage and to support conclusions.

Key takeaways

This project reinforced that good predictions often come from thoughtful feature design and careful diagnostics, not just fitting a model once. Addressing multicollinearity improved interpretability and helped avoid unstable coefficient behavior, making conclusions more reliable.
