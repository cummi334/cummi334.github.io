Project

Three-Body Orbital Dynamics Predictor

Built a Newtonian 3-body simulator to generate thousands of trajectories, label outcomes (stable/collision/ejection), engineer physics-based features, and train classifiers to predict long-term stability from initial conditions.

Python NumPy SciPy Simulation Dataset Generation Feature Engineering Classification Matplotlib
← Back to Projects Open PDF in new tab

What I did

• Implemented a numerical Newtonian gravity simulator and collected trajectory data across many randomized initial conditions.
• Labeled each simulation outcome as stable, collision, or ejection using rule-based criteria from the dynamics.
• Engineered physics-informed features (energy, angular momentum, barycenter-normalized coordinates) to make the dataset model-ready.
• Trained and evaluated classification models to predict long-term stability directly from initial conditions.

Results / Takeaways

This project taught me how to turn a simulation into a machine-learning dataset: defining labels carefully, choosing features that preserve invariants, and validating behavior with 2D/3D visualizations. Next improvements include more robust integrators, stratified sampling of initial conditions, and calibrated probability outputs.
